"save" will save the neural network of the current furthermost car, and this car will use to generate the next generation of cars

"refresh" will reload the page and restart the simulation with a new generation descending from the latest saved car

"delete" will delete from local storage the currently saved car and the simulation will start back to square one

"Number of cars" is the number of ai-controlled blue cars

"Mutation rate" controls the amount of differences that each generation will have compared to the currently saved best car; if the mutation rate is higher, every generation will differ more
